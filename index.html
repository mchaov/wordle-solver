SCORE 1 => scored based on letter frequency in the remaining set of words based on unique letters in the word

SCORE 2 => scored based on language letter frequency

SCORE 3 => scored based on letter frequency in the remaining set of words

9999 => the color of the column used for sorting

9999 => color of reference from other scoring methods

WORD => word exist in more than one of the shortlisted sets

Public API code example:

fetch("/parse", {
    method: "POST",
    body: JSON.stringify({
        lang: "en",
        pattern: "c????",
        letters: ["c", "a"],
        ignore: ["b", "x", "z"],
        unknowns: ["???a?"]
    })
})
.then(x => x.json())
.then(console.log)
